# Portfolio Project Descriptions

## Level 1: RESTful API with MongoDB

**Codveda Level 1: User Management API**

A foundational RESTful API built with Express.js and MongoDB, featuring user authentication, CRUD operations, and secure data management. This project demonstrates core backend development skills including database modeling, API design, and authentication middleware.

**Tech Stack**
- Node.js / Express.js
- MongoDB with Mongoose ODM
- JWT Authentication
- bcryptjs for password hashing
- REST API Architecture

**Key Features**
- User registration and login with JWT-based authentication
- Secure password storage using bcrypt
- CRUD operations for user management
- MongoDB database with proper indexing
- Input validation and error handling
- Environment-based configuration

**Live Demo**
- Backend: https://codveda-level1.onrender.com
- Frontend: https://codveda-level1.vercel.app

## Level 2: Full-Stack Authentication Dashboard

**Codveda Level 2: Admin Dashboard with User Management**

A complete full-stack application featuring role-based authentication, admin dashboard, and user management capabilities. Built with React and Express, this project showcases end-to-end development from database design to responsive UI implementation.

**Tech Stack**
- Frontend: React + Vite
- Backend: Node.js / Express.js
- Database: MongoDB Atlas
- Authentication: JWT + bcryptjs
- Styling: CSS with custom design system
- API: RESTful architecture

**Key Features**
- Secure authentication with role-based access control (Admin/User)
- Admin dashboard with user statistics and management
- Real-time user status tracking (online/offline)
- Profile management with editable fields
- User CRUD operations with permissions
- Responsive design with dark/light theme
- Protected routes and middleware
- MongoDB Atlas cloud database integration

**Live Demo**
- Backend: https://codveda-level2.onrender.com
- Frontend: https://codveda-level2.vercel.app

## Level 3: Real-Time Chat Application with GraphQL

**Codveda Level 3: Advanced Chat Platform with GraphQL & WebSockets**

A sophisticated real-time chat application featuring GraphQL API, WebSocket subscriptions, and advanced messaging capabilities. This project demonstrates mastery of modern full-stack development including real-time communication, GraphQL schema design, and premium UI components.

**Tech Stack**
- Frontend: React + Vite
- Backend: Node.js / Express.js
- API: GraphQL (Apollo Server)
- Real-time: WebSockets (Socket.io) + GraphQL Subscriptions
- Database: MongoDB Atlas
- Authentication: JWT
- State Management: Apollo Client
- UI: Custom components with premium design system

**Key Features**
- GraphQL API with schema-first design
- Real-time messaging with WebSocket subscriptions
- User presence tracking (online/offline status)
- Typing indicators and read receipts
- Message edit, delete, and copy functionality
- Premium UI with custom modals and animations
- User search and conversation management
- Role-based access control
- Optimistic UI updates with cache management
- Real-time notifications system

**Live Demo**
- Backend: https://codveda-level3.onrender.com
- Frontend: https://codveda-level3.vercel.app

## Project Overview

This three-level progressive project demonstrates growth from basic API development to advanced full-stack applications:

**Level 1** - Foundation: RESTful API with MongoDB, JWT authentication, and CRUD operations
**Level 2** - Full-Stack: React frontend with admin dashboard, role-based access, and user management
**Level 3** - Advanced: GraphQL API, real-time WebSockets, premium UI, and advanced messaging features

Each level builds upon the previous one, showcasing progressive skill development in:
- Backend API design (REST → GraphQL)
- Frontend architecture (Static → React → React + Apollo)
- Real-time communication (None → None → WebSockets)
- Database operations (Basic CRUD → Advanced relationships → Real-time subscriptions)
- UI/UX design (Basic → Responsive → Premium with animations)
